# myongstagram

### 개발 범위:

- 클라이언트: React Native
- 서버: Node.JS + Express + Postgres or MySQL

### 기본 기능:

- post: 글 혹은 사진을 작성
- 글목록
- 검색
- 프로필
- React Navigation을 활용한 UI
- Instagram과 유사하게 UI를 만들기

### 추가 기능:

- 사진 업로드
- 사용자 인증(토큰기반)
- 딥링크를 통한 Share
- Push Notification
